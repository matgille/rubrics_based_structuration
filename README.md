# Structuration unidocuments


Ce dépôt renferme les scripts permettant de structurer automatiquement un document pré-encodé en XML-TEI et d'en identifier les divisions principales. Il s'appuie essentiellement sur la typologie des lignes et l'identification des lignes rubriquées (en utilisant la classe `HeadingLine:rubric` de SegmOnto), et ne peut fonctionner sans celles-ci.
